# Grok 4.5 发布：性能进第一梯队，价格却砍掉一半，马斯克说"Opus 级"

**Grok 4.5 发布了**,官方这次连"聊天机器人"这个标签都懒得贴——**直接说这是为写代码、跑长任务、干知识工作准备的模型**,还是跟 Cursor 一起练出来的。

![](https://files.mdnice.com/user/166515/f820d3c0-d23f-4c76-be40-858526f64766.png)

这事儿本身就挺有意思。以前提起 Grok,大家第一反应是马斯克、是 X、是敢说敢怼,编程能力从来不是它的标签。这回不一样,讨论全在围着"能不能干活"转。

**跟 Cursor 一起训练,是这次真正的关键**

官方发布页写得挺直白:**Grok 4.5 是跟 Cursor 联合发布的,训练数据里塞了万亿级别的 Cursor 真实使用数据**——开发者怎么改代码、Agent 怎么调工具、怎么从错误里爬出来,这些都喂进去了。

这跟以前"刷榜"的打法不是一回事。SWE-Bench、HumanEval 这些榜单再高,也比不上真实 IDE 场景里几百万开发者天天产生的交互数据值钱。Hacker News 上有条评论说得挺准:**这可能是 Cursor 手里最值钱的资产,比它的编辑器本身还值钱**。

当然争议也在这——用户改代码的时候,**有多少人真的清楚自己的操作被喂进了下一代模型**?企业客户尤其在意这个,**透明度和 opt-out 机制目前都还没讲清楚**。

![](https://files.mdnice.com/user/166515/2de51ce6-6d3e-4a35-8f29-bb07be2981e8.png)
  
**跑分:进了第一梯队,但不是天花板**

具体数字上,Grok 4.5 在 Terminal-Bench 2.1 上跑到 **83.3%**,跟 GPT-5.5(83.4%)、Fable 5(84.3%)基本贴身;SWE-Bench Pro 上是 64.7%,比 Opus 4.8 低一截,但甩开了 GPT-5.5。Artificial Analysis 给它的综合排名是**全网第 4**,输出速度 87.6 token/秒,上下文 50 万 token。

翻译成人话就是:**不是最强,但已经不是陪跑的了**。X 上不少人的反应是"没想到 Grok 这次真到了 Claude、GPT 这个档位"。

![](https://files.mdnice.com/user/166515/18faad32-2fcc-41ef-aa4a-9fc73d54ed5b.png)

**真正的杀招是价格**

**输入 2 美元/百万 token,输出 6 美元/百万 token**——这个价格摆出来才是这次真正的冲击。同样水平的模型,Opus 4.8 报价是 5/25 美元,GPT-5.5 也不便宜。**如果能力真能打平,价格却是零头**,企业客户没理由不认真算这笔账。

![](https://files.mdnice.com/user/166515/8658cdec-aa0e-4ef2-aa8c-444aa734fdbe.png)

马斯克自己在 X 上补了一刀:现在这版还没用上他们给 GB300 定制的推理软件,**速度以后可能还能翻倍,上下文窗口也快要冲到 100 万 token**了。这话听听就好,但方向是清楚的——**xAI 这次是拿"模型+工具+算力+价格"打组合拳,不是单靠一个跑分数字**。

![](https://files.mdnice.com/user/166515/cb33e2ee-37a8-4bad-8023-d17198f0e78a.png)

**实测反馈:真项目里能干活**

开发者反馈里比较有代表性的是两类:一是有人拿它去优化 oxlint,**一次就挑出十几个性能问题**;二是有人把自己放了好几年、依赖全乱套的老项目丢给它,**居然被重新跑起来了**。这种"读懂一个陌生烂摊子"的能力,恰恰是真实工程里最难的部分,比写个 demo 有说服力得多。

但也有人提到,**长对话跑到后半段,Grok 4.5 有点"变懒"的迹象**——响应变敷衍,这个还需要更多人验证。

**信任问题绕不开**

Hacker News 那条讨论下面几百条评论,**吵得最凶的不是跑分,是信任**。Grok 过去在政治立场、内容尺度上的争议不少,不少人说这让他们很难把它放进正经的商业场景里用。也有反方观点:所有模型厂商都会通过系统提示词和后训练去塑造输出风格,区别只是做得明不明显。

![](https://files.mdnice.com/user/166515/6aa21756-76d2-4c59-8b5e-2769a09b3f97.png)

这话说得也在理——但对企业用户来说,**"透明"和"可控"从来都比"聪明"更重要**。

日常聊天写东西,Grok 4.5 不一定马上能取代你手里的 GPT 或 Claude。但如果你在 Cursor 里写代码、跑 Agent 改多文件项目、或者就是单纯想找个便宜点的 API 备选,**这次值得试试**——尤其 Cursor 用户,**首周还有双倍额度**。
